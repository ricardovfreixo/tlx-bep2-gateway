{{chainName}} <-> Turtle Network Gateway

This is a simple gateway for ERC-20 tokens to the TN Platform. It is based on tunnels. Users can define a tunnel between a source ETH address and a target TN address. Whenever an ERC-20 token is send from the defined source ETH address to the gateway address, the corresponding amount of tokens are send to the defined TN address on the TN Platform. It is important to notice that tunnels will be removed after they are used. Therefore, if more than one transfer should be done via a tunnel, this tunnel needs to be re-established!

For sending tokens from the TN Platform to the Ethereum blockchain, just add the ETH address that should receive the tokens as the description of the transfer and send the tokens to the TN address of the gateway.
